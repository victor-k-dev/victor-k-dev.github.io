My Work

Current and past projects.

s3bfd

s3bfd is an application designed to download files from S3 buckets. It can be run from the command-line or via the GUI, create a local cache of a bucket's metadata, and validate downloaded files. All of the files in a given bucket can be downloaded, or a specific prefix (i.e. folder) can be targeted.
This project is currently under active development, and continues to receive improvements and new features.

live-server

A simple, GUI-based live server for web development. It's written in Python for cross-platform compatibility, and auto-reloads the web page as it's modified, allowing changes to the page to be reflected immediately. It's a stand-alone application, so it can be used independently of an IDE.
